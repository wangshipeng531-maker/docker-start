# Docker 入门完全指南
一篇面向零基础读者的 Docker 入门教程，内容涵盖理论与实战，覆盖 Windows 环境下的完整安装流程、常用命令、镜像原理、数据卷、Dockerfile 及网络配置。
## 目录
- Docker 核心概念（镜像、容器、仓库）
- Docker 与虚拟机对比
- Windows 安装 Docker Desktop 全流程
- 国内镜像加速配置
- 常用命令速查（镜像 / 容器 / 数据卷）
- 实战部署 Nginx + MySQL
- Docker 镜像分层原理
- 容器数据卷与数据持久化
- Dockerfile 编写与自定义镜像构建
- Docker 网络配置
- 常见问题速查
## 适用读者
- 初次接触容器技术的开发者
- 需要在 Windows 上快速搭建 Docker 环境的用户
- 想系统掌握 Docker 基础知识的工程师
